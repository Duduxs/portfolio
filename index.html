Sobre mim

Quem é Eduardo José?

Sou um desenvolvedor back-end e venho estudando a stack do Java EE Web, criando projetos diversos do seu ecossistema.

No 4º período do curso de Análise e Desenvolvimento de Sistemas da universidade IESO.

Anteriormente, eu tive experiência trabalhando com um grupo indie de Jogos Digitais.

Eu estava trabalhando como programador, desenvolvendo jogos para mobile, desktop e web. Entretanto, eu decidir focar na criação de softwares no geral.

Tecnologias

Confira as tecnologias que eu mais utilizo

Java

Uma das linguagens de programação mais usada no mundo.

Spring Framework

O framework mais popular do Java, tem uma programação compreensiva e modelos de configuração para Java EE.

Jpa

É a especificação padrão da plataforma Java EE para mapeamento objeto-relacional e persistência de dados.

Hibernate

Framework ORM que segue as especificações do JPA, é a implementação física persistir, remover, atualizar ou buscar dados.

JavaFX

Plataforma de software multimídia para a criação e disponibilização de Aplicação Rica para internet que pode ser executada em vários dispositivos diferentes.

JUnit

Framework com suporte à criação de testes automatizados, facilitando a criação e manunteção do código.

SQL

Linguagem de pesquisa declarativa padrão para bancos de dados relacionais.

NoSQL

Linguagem que representa os bancos de dados não relacionais.
