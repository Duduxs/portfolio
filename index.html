Sobre mim

Quem é Eduardo José?

Sou um desenvolvedor back-end e venho estudando a stack do Java EE Web, criando projetos diversos do seu ecossistema. E, no momento, estou trabalhando como desenvolvedor na empresa Compasso UOL. Atualmente, graduado no curso de Análise e Desenvolvimento de Sistemas da faculdade IESO, e com pós-graduação de administração de bancos de dados na Faculdade Metropolitana da Grande Recife.

Anteriormente, eu estive com um grupo indie de Jogos Digitais estando na equipe como programador. Eu desenvolvia jogos para mobile, desktop e web, entretanto, por motivos de mercado deixei essa área de lado para focar no desenvolvimento de aplicações web, no qual contribuo profissionalmente a mais de 3 anos.

Também já trabalhei como programador júnior fullstack na e-Precise, atuando com Java EE, Kotlin, Mongo, Angular e muitas outras tecnologias.

Estou silenciosamente confiante, naturalmente curioso e perpetuamente trabalhando para enfrentar todos os desafios que venham pela frente.

Tecnologias

Confira as tecnologias que eu mais utilizo

Java & Kotlin

Java é uma das linguagens mais usadas do mundo e o Kotlin promove uma grande modernidade, flexibilidade e manutenibilidade de código, além de ser interoperável com o Java.

Spring

O framework mais popular do Java, tem uma programação compreensiva e modelos de configuração para Java EE.

Quarkus

Framework Java nativo, onde oferece pequeno consumo de memória e tempo de inicialização reduzido.

Docker

É uma plataforma de software que permite a criação, o teste e a implantação de aplicações rapidamente através de contêineres.

Angular

Framework frontend baseado em typescript para criação de SPAs & PWAs, e também facilita a criação de GUIs no geral.

Kafka

Plataforma distribuída de transmissão de dados que é capaz de publicar, subscrever, armazenar e processar fluxos de informações em real time.

SQL

Linguagem de pesquisa declarativa padrão para bancos de dados relacionais.

NoSQL

Linguagem que representa os bancos de dados não relacionais.
